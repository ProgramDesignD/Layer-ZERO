# Layer-ZERO

## Install
(python 3.10 or higher is required)
```bash
pip3 install poetry
git clone https://github.com/ProgramDesignD/Layer-ZERO
cd Layer-ZERO/
poetry install
```
## Commit
### First time
```bash
git branch --set-upstream-to origin/main
```
### Every time
```bash
git pull
git commit -m "commit message"
git push
```
